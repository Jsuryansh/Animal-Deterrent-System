- Arduino Uno

## AI Processing Device

- Android Smartphone
- OnePlus 9R (Test Device)

## Servos

- MG996R Pan Servo ×1
- MG996R Tilt Servo ×2

- USB OTG Adapter
- USB Cable

- External 5V Supply

- L298N Motor Driver
- DC Motor ×2
