# AI Model Information

## Model

EfficientDet Lite0

## Framework

TensorFlow Lite

## Input

RGB Camera Frames

## Output

Bounding Boxes
Class Labels
Confidence Scores

## Supported Classes

- Cow
- Dog
- Cat
- Horse
- Sheep
- Bird
- Elephant
- Bear
- Zebra
- Giraffe

## Custom Model Support

The system supports custom TensorFlow Lite models.

Examples:

- Goat
- Monkey
- Deer
- Buffalo
- Wild Animals
